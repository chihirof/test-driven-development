# 『テスト駆動開発』 写経用リポジトリ

## 環境構築

### Typescript

https://typescript-jp.gitbook.io/deep-dive/getting-started

  1. $ npm init  
  1. $ npm install typescript -D
  1. `package.json` に以下を追加

      "scripts": {
        "tsc": "tsc"
      }

  1. $ npx tsc --init
